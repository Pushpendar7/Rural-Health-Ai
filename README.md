# 🏥 Multilingual Rural Health Assistant
An AI-powered prototype built for the **Solution Challenge 2026**.

## 🌟 Overview
This tool helps patients in rural areas understand complex medical prescriptions by using AI to read handwriting and explain medicines in regional languages.

## 🛠️ Tech Stack
- **AI Model:** Google Gemini 2.5 Flash (Multimodal)
- **Frontend:** Streamlit
- **Language:** Python
- **Environment:** Google Cloud / AI Studio
